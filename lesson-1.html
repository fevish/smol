The Basics

In this first lesson, we will learn how to use simple text editor tools on your computer to create the most basic type of website. In future lessons we will be upgrading to more advanced (and free to use) tools, but for now these will be sufficient.

All of these lessons are meant to be completed on a computer running any version of Windows or Mac OS.

Folder and Text Editor Setup

For the purpose of these lessons, we are going to assume the user or user's supervisor (parent, sibling, teacher etc.) has a basic understanding on using their computer's operating system.

To begin, we are going to create a new folder on our desktop and call it HTML Lessons.

Next, we will open up our text editor. If you are using Windows, open Notepad. You can accomplish this by typing Notepad into your search bar to open a blank notepad. On Mac, the equivalent is called TextEdit.

Once your text editor is open, you should see something similar to this:

Now click File > Save As.. and save the file name as index.html in the folder we created earlier called HTML Lessons. Make sure you indicate Save as type: to All Files (*.*).

You should now see your new HTML file in your lessons folder. Go ahead and open that file. By default the file should open up in your web browser (Chrome, Edge or Safari etc.). This is because this is an HTML file, which is a web page file.

You probably noticed that your file loaded to a blank white page. We haven't put any code or content in it yet.. let's fix that!

Keep your web browser open. Minimize it or resize it to fit half your screen.

Hello World!

Time to write our first block of HTML code. Right now, when we opened up our HTML file it loaded in our default web browser. We must also open our file in our text editor to edit it. Navigate to Desktop > HTML Lessons > index.html and right click on our index.html file and click Open with > Notepad. If you do not see Notepad in the list, then select Select another App and locate Notepad.

Markup

With both your index.html web page and your text editor open, let's add some proper elements, tags and markup. First we will add our HTML and BODY elements to our code. Update and save your code with the following:

{{ site.code }}

Hello World!

This is my first website.

{{ site.endcode }}

Result:

Hello World!

This is my first website.

Look back at your code. Most HTML elements require opening < > and closing < /> tags.

  • <html> is the root element of an HTML page. This element tells the browser that this is an HTML document and is the container for nearly all other HTML elements. The html element has opening <html> and closing </html> tags. Notice the order of our elements, how they cascade to mirror themselves, and close with / appended to the element name.
  • <body> is where all of our webpage content lives. Everything between the opening and closing body tags will be seen visually by the viewer (text, images, etc.).
  • <h1> is a heading element. It is used for large heading text to capture attention. Heading elements need opening <h1> and closing </h1> tags.
  • <p> is a paragraph element. It is used for most written text in websites. By now you probably are noticing that most elements require closing </ > tags.

Modify your code and remove the closing </h1> tag, then refresh your page and see the result.

{{ site.code }}

Hello World! -

This is my first website.

{{ site.endcode }}

Result:

Hello World!

This is my first website.

Closing your tags is important! If you forget to close a tag, it is often VERY noticable on your webpage.

Add a splash of color

Now that we have created a basic web page, we are going to learn some styling basics. There are two primary ways to style your website: stylesheets and inline styles. We will learn about stylesheets in another lesson, for now we will stick to inline styles.

Inline styling is the most basic way to style elements in a website. Let's go back to our code and fix our closing H1 tag, and add a bit of color. Add the following to your code:

{{ site.code }}

Hello World!

I am smol.

This is my first website.

{{ site.endcode }}

Result:

Hello World!

I am smol.

This is my first website.

It may not look elegant yet, but you've just learned how to paint with code! Imagine what you can do once you learn some more styles!

  • style=" " This tells the tag that we are applying inline styling to it. Notice the opening and closing apostrophes.
  • color: red; This is a declaration. We have two parts to our declaration: Propert and Value. color is the property, and red is the value.
  • font-size: 50px; font-size is a property, and 50px is a value. px stands for pixels. Notice the semicolon ; at the end of our declarations. Make sure you always end a declaration with a semicolon.
  • Did you notice that our inline styling ONLY APPLIES to the single <h1> element it was applied to? If you want to apply a style to multiple elements, you can do this with stylesheets which we will learn about in the next lesson!

Closing Thoughts

Whoa, that's a great start! We just learned how to create a local web page, and the basic building blocks of HTML.

Go ahead and play with your code; try adding some styles to the other elements and see what happens. Here are a few new styles to try out and see what happens (try different values):

font-style: bold, italic, underline;
background-color: yellow;
text-align: center, left, right;
